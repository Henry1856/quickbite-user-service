# QuickBite Roles System

## Roles

### customer
- Browse restaurants
- Place orders
- Track deliveries

### owner
- Manage restaurant
- Manage menu
- View restaurant orders

### driver
- Track deliveries
- Update delivery status/location

### admin
- Full system access
- Manage all services
- Monitor platform activity

---

### guest (unauthenticated user)
-Read-only public acess only
